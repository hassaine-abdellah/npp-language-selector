Language Selector (plugin for Notepad++)

This Plugin was made to be as simple as possible, and as usefull -for its purpose- as possible:
It  allows you to quickly select the programming language for the current tab, and by consequence apply the corresponding syntax highlighting.
Features:
• Select programming language from dropdown menu.
• Auto-show selection dialog on new tabs (can be turned off).
• Pin favorite languages on top of the list for quick access (when favorites are full, the oldest one (first in) is put out).
• Configurable favorite list size.
